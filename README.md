**Rune Breaker**

Rune Breaker is a brick-breaking arcade game, but with rune-themed bricks, power-ups, and dynamic effects.
Control the paddle to bounce one or more balls and break rune-imbued bricks.
Use power-ups strategically to survive and maximise your score.
Destroy all bricks to advance to the next level.

**Controls**
Move Paddle: Left / Right arrows
Launch Ball: Spacebar
Laser Paddle: Fire lasers at bricks (power-up)
Pause Game: P or Esc

**Power-ups**

Multi-Ball: Spawn extra balls
Wide Paddle: Increase paddle width
Slow Ball: Reduce ball speed
Extra Life: Gain one life
Laser: Activate laser firing
Sticky: Stick balls to the paddle

AI Usage I used AI (ChatGPT) to help make the rune system and menu animations. This includes rune shapes, the font mapping, and the drop shadow text. I checked and edited all the code myself.
